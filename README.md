Овој проект претставува анализа на нивото на стрес кај студенти користејќи податочни сетови `StressLevelDataset.csv` и `Stress_Dataset.csv`. Целта е да се истражи како различни фактори како анксиозност, самодоверба, квалитет на сон, социјална поддршка и академски оптоварувања влијаат врз вкупното ниво на стрес.
